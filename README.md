# Connect Advisor — in-browser demo

A self-contained, in-browser prototype of an agentic new-customer-setup scoping tool:
a branched scoping interview that, on explicit human sign-off (HITL gate), generates a
full setup bundle.

**Live demo:** https://chesler410.github.io/connect-advisor-demo/

No backend — everything runs in the page (`docs/index.html`). Sample artifacts use the
fictional **"Acme Industries"**; no confidential data.
